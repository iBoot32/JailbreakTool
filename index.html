JailbreakTool

A work-in-progress jailbreak for iOS 6.1.6 among other versions

How to run: download https://raw.githubusercontent.com/iBoot32/JailbreakTool/master/prep.bat and run it with your iPhone 3GS connected in DFU mode

IMPORTANT: THIS TOOL, DUE TO ITS RELIANCE ON LIMERA1N, WILL ONLY WORK IN WINDOWS 7!!! LIMERA1N IS CURRENTLY BROKEN IN ANY NEWER WINDOWS VERSION!!!
